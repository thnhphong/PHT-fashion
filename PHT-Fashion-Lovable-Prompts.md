## GLOBAL DESIGN GUIDELINES

**Typography:**
- Headings: Bold, modern sans-serif (Montserrat, Inter, or Poppins)
- Body: Clean sans-serif (Inter, DM Sans)
- Accent text: Mix serif for elegance on specific sections

**Color Palette:**
- Primary: #000000 (Black)
- Secondary: #1a1a1a (Dark Gray)
- Accent: #FF1744 (Neon Pink) or #00E5FF (Electric Blue)
- Success: #00C853 (Green)
- Background: #FFFFFF (White) and #F5F5F5 (Light Gray)
- Text: #212121 (Dark) and #757575 (Gray)

**Spacing:**
- Consistent 8px grid system
- Generous padding on sections (80px-120px vertical)
- Card spacing: 24px-32px gaps

**Buttons:**
- Primary: Black background, white text, rounded
- Secondary: Outlined, transparent background
- Hover: Subtle scale up (1.02-1.05) with shadow
- Active: Slight press-down effect

**Cards:**
- Border radius: 12px-16px
- Shadow: subtle, layered shadows
- Hover: lift effect with increased shadow
- Transition: 0.3s ease

**Animations:**
- Fade-in on scroll
- Slide-up for sections
- Scale for buttons
- Smooth page transitions
- Loading skeletons, not spinners

**Icons:**
- Use Lucide React or Heroicons
- Consistent stroke width
- Meaningful, recognizable icons

**Responsive Breakpoints:**
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

**Accessibility:**
- WCAG AA compliant
- Keyboard navigation
- Screen reader friendly
- Focus states visible
- Alt text on images
