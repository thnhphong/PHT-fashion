AUTH
POST /api/register (register new user account)
POST /api/login (authenticate user and return JWT)
POST /api/logout (logout user, invalidate token on client)
PRODUCTS
GET /api/products (list products with optional filters)
GET /api/products/{id} (get product details by id)
CATEGORIES
GET /api/categories (list all product categories)
CART (AUTH)
GET /api/cart (retrieve current user cart)
POST /api/cart/add (add product to cart)
DELETE /api/cart/remove/{itemId} (remove cart item)
CHECKOUT (AUTH)
POST /api/checkout (create order from cart and apply coupon)
ORDERS (AUTH)
GET /api/orders (get authenticated user order history)
GET /api/orders/{id} (get order details)
COUPONS
POST /api/coupons/validate (validate coupon code and calculate discount)
PROFILE (AUTH)
GET /api/profile (get authenticated user profile)
PUT /api/profile (update authenticated user profile)

ADMIN AUTH
POST /api/admin/login (authenticate admin and return JWT)
POST /api/admin/logout (logout admin)
ADMIN DASHBOARD (ADMIN)
GET /api/admin/dashboard (get system statistics overview)
ADMIN USERS (ADMIN)
GET /api/admin/users (list all users)
PUT /api/admin/users/{id}/role (update user role)
DELETE /api/admin/users/{id} (delete user)
ADMIN PRODUCTS (ADMIN)
POST /api/admin/products (create new product)
PUT /api/admin/products/{id} (update product)
DELETE /api/admin/products/{id} (delete product)
ADMIN CATEGORIES (ADMIN)
POST /api/admin/categories (create new category)
DELETE /api/admin/categories/{id} (delete category)
ADMIN ORDERS (ADMIN)
GET /api/admin/orders (list all orders)
PUT /api/admin/orders/{id}/status (update order status)
ADMIN COUPONS (ADMIN)
POST /api/admin/coupons (create coupon)
DELETE /api/admin/coupons/{id} (delete coupon)
NOTES
AUTH = requires valid JWT
ADMIN = requires JWT with admin role
Logout handled client-side by removing JWT (optionally server blacklist)
